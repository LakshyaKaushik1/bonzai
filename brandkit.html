6. Signature glyphs

A tiered system: Tier 1 renders correctly in any standard monospace font (Menlo, Consolas, SF Mono, DejaVu Sans Mono) — this is the safe default. Tier 2 is a richer treatment that only shows if a Nerd Font is detected on the user's system, with automatic fallback to Tier 1.

Tier 1 — universal fallback (default, always safe)

Element Glyph Codepoint Why this one
Active prompt ❯ U+276F Heavy single chevron — standard in Fish/Starship, wide font support, distinct from plain >
Prompt (busy/thinking variant) ◆ U+25C6 Solid diamond — reads as "agent turn," pairs with rose accent
Success ✓ U+2713 Universally supported check mark
Error ✕ U+2715 Cleaner and bolder than a plain x, still ubiquitous
Warning ⚠ U+26A0 Standard warning triangle, wide support
Info / tool call ℹ U+2139 Info symbol, safe fallback
Running / spinner frame ⠋⠙⟹⟸ U+2800 block Braille-pattern spinner — same trick used by most modern CLIs (npm, cargo), animates smoothly, near-universal font support
Queued / pending ◷ U+25F7 Clock-quadrant, safe fallback
Cancelled ⊘ U+2298 Circle-slash, unambiguous
Diff added + U+002B Kept as plain + — over-styling diffs hurts scanability
Diff removed - U+002D Plain -, same reasoning
Section divider ────── U+2500 Box-drawing light horizontal, safe in all monospace fonts
File path marker › U+203A Single guillemet, reads as breadcrumb separator

Tier 2 — enhanced (Nerd Font detected)

Element Use case Nerd Font glyph name Notes
Active prompt Rounded arrow divider nf-pl-right_soft_divider Powerline-style rounded arrow — Starship/Powerlevel10k standard
Brand mark (footer/banner) Bird-adjacent icon nf-md-bird Consider commissioning a genuinely custom parakeet PUA glyph long-term — see note below
Success Filled check circle nf-fa-check_circle
Error Filled x circle nf-fa-times_circle
Git branch (if shown in status bar) Branch icon nf-dev-git_branch
Detection tip: don't ship Tier 2 by default. Detect via an env var convention already used for this purpose ($NERD_FONT, the same one Starship/Oh-My-Posh check) or a one-time setup prompt ("does this render as a solid triangle? ▶") rather than assuming. Always fall back to Tier 1 if detection is inconclusive — a broken glyph on first launch is a worse first impression than a slightly plainer one.

Why not fully custom Private-Use-Area (PUA) glyphs

Nerd Fonts inject icons into Unicode's Private Use Area — special codepoints that only resolve into pictures if the right patched font is installed; without it they show as blank boxes. A fully custom parakeet glyph (e.g. a tiny parakeet head as your prompt icon) is possible but would require distributing your own patched font, which most users won't install just for one CLI tool. Recommended path: ship Tier 1 as default, Tier 2 using existing common Nerd Font icons (so users who already have one installed for Starship/vim get the upgrade for free), and treat a fully custom bird glyph as a "nice to have later" rather than a v1 requirement.
